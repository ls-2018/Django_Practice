#   第三方插件
    
    -   pip install virtualenv                  安装到系统目录
    -   pip install virtualenv  --user          安装到用户目录
    -   注意如果没有配置User PATH       需要在~/.bashrc中新增  export PATH="$HOME.local/bin:$PATH"
    -   virtualenv  project-env
    -   cd project-env
    -   source bin/active
    -   deactive
